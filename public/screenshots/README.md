## Tips para Mejores Screenshots

1. **Usa tema claro** - Se ve mejor en README
2. **Sin datos sensibles** - Usa datos de demo o borra información real
3. **Captura completa** - Incluye el header y sidebar si aplica
4. **Alta resolución** - Mínimo 1280x720
5. **Formato PNG** - Mejor calidad para README
6. **Evita elementos de navegador** - Oculta bookmarks o extensiones

## Cómo Tomar Screenshots

### Windows
- **Snipping Tool**: `Win + Shift + S`
- **Screenshot completo**: `Win + PrtScn`

### macOS
- **Screenshot**: `Cmd + Shift + 4`

### Chrome DevTools (opcional)
1. Presiona `F12`
2. Click en el ícono de dispositivo (Ctrl+Shift+M)
3. Selecciona "Responsive Design Mode"
4. Selecciona tamaño deseado
5. Toma screenshot del área específica
